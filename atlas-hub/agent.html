CV Screener
HR & recruiting GPT-4o CSV output ATS compatible batch processing
SA

Uploaded 12 May 2025 · People Ops @ Acme
Description
Ranks applicants against a job description using structured criteria. The agent parses each CV, scores it across required skills, experience level, and role alignment, then outputs a ranked shortlist as a CSV file. Flags mismatches with a reason column so reviewers can quickly audit decisions. Designed for high-volume screening where you need a first-pass filter before human review.
Constraints
Input format
PDF or plain-text CVs only. Does not process scanned images or LinkedIn exports.
Batch size
Tested up to 200 CVs per run. Performance degrades above 500.
When to use
Screening more than 20 applicants for a structured role with clear criteria.
When NOT to use
⚠ Roles requiring portfolio review, creative assessment, or senior leadership judgment. Do not use as the sole decision tool.
Model required
GPT-4o (function calling enabled). Will not run on GPT-3.5.
